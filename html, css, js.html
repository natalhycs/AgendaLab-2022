    La función que se crea o las instrucciones que quiero que se ejecute es el script para que tenga la capacidad de interacturar con el botón

    Spring Boot

    Framework, Spring Boot // Serie de bibliotecas, que puedo usar para agilizar un desarrollo, hace la configuración automática con un servidor web incluido dentro de mi aplicación, es capaz de publicar páginas web, almacenar la información.
    Es para hacer aplicaciones back end.
    Seleccionar proyecto, lenguaje, la última version de sb estable group es el dominio al reves artifact y name el mismo nombre de mi proyecto todo recomendado en minusculas.
    Cuando se termine la configuración del proyecto se crea una carpeta con todo lo que se requiere para ejecutar mi aplicación.
    y se pueden agregar dependencias desde vs para realizar lo que se requiera con la dirección y artifactid

    Carpetas

    • java: Todo el código fuente va en esta carpeta, todo lo compilable
    • resources: Todo los recuersos que no sean código fuente, estático.
    • Static: Todo lo que no necesito hacer nada img, video
    • Templates: Todas las plantillas con las que trabaje se dejan ahí, freemaker
    • Aplication.properties: se configuran los diferentes puertos para que no tengamos conflictos con otras aplicaciones se debe poner server.port = 8090, 8081, etc, etc
      • Para arrancar la aplicación el proceso es ir a java, abrir la aplicación y clic en run
